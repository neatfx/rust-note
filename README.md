---
文档位于 `src/` 目录，可使用 [mdbook](https://crates.io/crates/mdbook) 工具生成 HTML 格式：

```bash
# 第 1 步：安装文档构建工具
cargo install mdbook

# 第 2 步：渲染 Markdown 文件，输出 HTML 页面至 “book/” 目录
mdbook build

# 第 3 步：使用浏览器打开 "book/index.html" 查看文档
```
